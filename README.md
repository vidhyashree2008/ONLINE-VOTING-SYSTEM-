🗳️ Online Voting System (Frontend)

A simple Online Voting System built using HTML, CSS, and JavaScript.
This project allows users to vote for candidates and view live results instantly in the browser.

---

📌 Features

- ✅ Vote for multiple candidates
- ✅ Instant result display
- ✅ Simple and clean UI
- ✅ Fully browser-based (no backend required)

---

🛠️ Technologies Used

- HTML
- CSS
- JavaScript

---

📂 Project Structure

online-voting-system/
│
├── index.html
├── style.css
├── script.js
└── README.md

---

🚀 How to Run the Project

1. Download or clone this repository
2. Make sure all files are in the same folder
3. Open "index.html" in any web browser

---

⚠️ Limitations

- ❌ No user authentication
- ❌ Votes are not stored permanently
- ❌ Users can vote multiple times
- ❌ Not secure for real-world elections
